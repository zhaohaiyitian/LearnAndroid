








Room 包含三个主要组件：
数据实体 Entity：用于表示应用数据库中的表
数据访问对象 Dao：提供可用于查询、更新、插入和删除数据库中的数据的方法
数据库类 Database：用于保存数据库并作为应用持久性数据底层连接的主要访问点

数据库事务
1.安全性
可以确保数据的完整性和一致性
2.高效性

没开启事务时 打开数据库 插入数据库 关闭数据库----耗时很多
开启事务时 把数据存到内存里，然后一次性写入到数据库里


如何优化大量数据的插入操作？
1.使用事务
2.批量插入 合并插入语句
3.使用预编译语句  预处理语句是指在执行前被编译并优化的 SQL 语句。这种方式可以提高插入性能，因为它减少了 SQL 语句解析的开销。

如何优化大数据量的查询性能？
1.使用索引
2.使用预编译语句
